# Type Resume

[View Site](http://www.ankush.tech/resumex)

## Building

```bash
git clone git@github.com/ScorchingShade/ResumeX.git
cd ResumeX
npm install
npm run dev
# Open localhost:4003/index-dev.html in your browser
```

## Building for Production

```bash
npm run build
```
